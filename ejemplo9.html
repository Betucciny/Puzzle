Este proyecto es un puzzle deslizante 3x3, utiliza un método de búsqueda para encontrar la forma de resolverlo, ejecutando esta solución para mostrarlo al usuario. Los métodos de búsqueda son una técnica en inteligencia artificial para encontrar una respuesta, balanceando lo rápido que la encuentra y lo buena que es esta respuesta. La interfaz se realizó con HTML, CSS y JavaScript, herramientas que permiten la creación de interfaces de usuario amigables.
