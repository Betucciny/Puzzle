Este proyecto es el juego del gato, utiliza el algoritmo Minimax como técnica de Inteligencia Artificial para elegir el mejor movimiento. El algoritmo MiniMax sirve para elegir una jugada óptima, entre todas las disponibles, para un estado de la partida en concreto. La interfaz se realizó con HTML, CSS y JavaScript, herramientas que permiten la creación de interfaces de usuario amigables.
